# Changelog

## v0.1.0 - 2026-05-09

### Added

- Initial local routing CLI with `classify`, `route`, and `explain`.
- Provider profiles and cost, risk, complexity, and latency scoring.
- Dry-run route records with local JSONL history.
- Routing policy documentation and example configs.

### Security

- No API keys are required for dry-run routing.
- No live provider execution is performed by the MVP.
